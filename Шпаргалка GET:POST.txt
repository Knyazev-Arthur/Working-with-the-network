Отправка GET запроса
1. Создание URL: Создать объект URL, указывающий на ресурс, который мы хотим получить.
2. Создание запроса: Создать объект URLRequest (HTTP запрос с параметрами) с указанием метода GET и URL.
3. Отправка запроса: Создать объект URLSession (сессия запроса(выполнение запроса, обработка данных)) и выполнить запрос с помощью метода dataTask.
4. Обработка ответа: В блоке завершения dataTask вызвать метод для декодирования полученных данных в нужный формат, обработать код состояния HTTP и возможные ошибки.

Отправка POST запроса
1. Создание URL и параметров запроса: Создать URL и определить параметры (словарь/тело(body)) для отправки на сервер.
2. Создание запроса: Создать объект URLRequest, создать заголовок Content-Type, вызвать метод для кодирования отправляемых данных в нужный формат, установить метод POST и добавить параметры запроса в тело запроса.
3. Отправка запроса: Создать объект URLSession и выполнить запрос с помощью метода dataTask.
4. Обработка ответа: В блоке завершения dataTask обработать полученные данные, код состояния HTTP и возможные ошибки.
